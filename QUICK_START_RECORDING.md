# 🎬 Quick Demo Recording - Start Here

## ⚡ Fastest Way (2 minutes)

### **Step 1: Install Requirements**
```bash
pip install mss opencv-python
```

### **Step 2: Run Recorder**
```bash
python simple_recorder.py
```

### **Step 3: Follow Prompts**
- Press Enter when ready
- Perform your app demo for 3 minutes
- Video saves automatically as `ComDetect_Demo.mp4`

## 🎯 Pro Tips

✅ **DO:**
- Move mouse slowly and deliberately
- Pause 2-3 seconds between actions
- Click buttons clearly and wait for response
- Scroll smoothly
- Give app time to load/process

❌ **DON'T:**
- Rush through the demo
- Click too fast
- Open other windows/apps
- Have background motion/noise

## 🆘 Troubleshooting

| Problem | Solution |
|---------|----------|
| "Camera not found" | Check camera permissions, test with Windows Camera app |
| "App won't start" | Run `streamlit run app.py` manually first to test |
| "Recording is laggy" | Close other apps, try 24 FPS instead of 30 |
| "Face detection fails" | Ensure good lighting, sit 30cm from camera |
